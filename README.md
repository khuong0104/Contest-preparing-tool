# Contest-preparing-tool
### For C++ users by far :))

***Features:***
- Create folder and file for each problem

### How to use:
1. Download this repository.
2. Open **Command Prompt**.
3. Change the directory to the folder **Contest-preparing-tool**.
4. Add your snippet 
5. Run `py ContestPreparingTool.py`.
